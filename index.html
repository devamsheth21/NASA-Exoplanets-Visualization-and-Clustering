Planet Aquarium

From the NASA Exoplanet dataset, 4 clusters of planets were selected using clustering algorithms to plot those planets around their common center(sun). The planets are visualized in form of Sea creatures,where different planet types are shown using different seacreatures. The brightness(stellar magnitude) is visualized using color. The radius of the planet is represented through size of each planet(sea-creature). The user can hover on each planet to see their orbit, and other information like Discovery year,Name, Mass and other attributes. The legend shows the types of planets along with a count of how many of each type are in the visualization. A bar chart alongside the legend shows a horizontal bar for each planet type, scaled according to the count of planets of that type in the visualization.The cluster select option allows the user to filter the visualization by different groupings of planets.
